# FF-Mogno

#### FF-Mogno é um Tema que lembra a Madeira: Mogno. É legível, quase Dark e muito atrativo aos olhos. (FF - Felipe Facundes). ####

<br></br>

### Instalação:

##### Icones:
```
cd Icones/
7z x FF*7z
cp -rf FF-Mogno ~/.icons/
```
<br></br>

##### GTK:

```
cd GTK/
cp -rf FF-Mogno ~/.themes/
